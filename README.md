## Prerequiresites
- `docker`
- `nodejs`
- `minikube` / `kind`

<br />

How to setup the project repos?

- Create a new folder with any name

- Go inside of that folder

- Open a new terminal or CMD

- Clone the main repo
```bash
git clone 
```

- Clone frontend project
```bash
git clone https://github.com/JeralSandeeptha/sitecore-genai-frontend
```

- Clone gateway service project
```bash
git clone https://github.com/JeralSandeeptha/sitecore-genai-gateway
```

- Clone user service project
```bash
git clone https://github.com/JeralSandeeptha/sitecore-genai-users
```

- Clone url generation service project
```bash
git clone 
```

- Clone infrastructure project
```bash
git clone https://github.com/JeralSandeeptha/sitecore-genai-infra
```

- Clone kubernates project
```bash
git clone https://github.com/JeralSandeeptha/sitecore-genai-k8s
```

- Clone dev environment project
```bash
git clone https://github.com/JeralSandeeptha/sitecore-genai-development
```

- Clone project architecture
```bash
git clone https://github.com/JeralSandeeptha/sitecore-genai-architecture
```

- You can run projects through one by one or you can run the whole project using docker compose which is the file lives under develop folder in `sitecore-genai-development` repo.
