## Prerequiresites
- `docker`
- `nodejs`
- `minikube` / `kind`

<br />

How to setup the project repos?

- Create a new folder with any name

- Go inside of that folder

- Open a new terminal or CMD inside of that folder on `Windows`. If you are using `other` OS types you can clone these all projects under the folder you created

- Clone the main repo
```bash
git clone https://github.com/JeralSandeeptha/sitecore-genai-project
```

- Clone the sitecore data repo
```bash
git clone https://github.com/JeralSandeeptha/sitecore-genai-data
```

- Clone frontend project
```bash
git clone https://github.com/JeralSandeeptha/sitecore-genai-frontend
```

- Clone gateway service project
```bash
git clone https://github.com/JeralSandeeptha/sitecore-genai-gateway
```

- Clone user service project
```bash

```

- Clone AI service project

- Clone Code generator service project
